    Projetos

    Projeto Sama

    O projeto Sama foi desenvolvido por mim e pela minha equipe de projetos integrados ".Tec". O projeto consiste em um sistema de gerenciamento de atestados e equipes, onde o usuário pode enviar atestados, editar editar e gerenciar equipes. O sistema foi desenvolvido utilizando HTML, CSS, JavaScript e Python com o framework Flask. Veja o repositório no GitHub.

    O projeto foi desenvolvido em equipe, onde cada membro teve um papel importante na construção do sistema.

    Minhas Contribuições

    Eu fui responsável pelo desenvolvimento do front-end, onde utilizei HTML, CSS e JavaScript para criar uma interface amigável e intuitiva para o usuário..

    Desafios Enfrentados

    Participei do desenvolvimento do projeto Sama como desenvolvedor front-end, sendo meu primeiro contato prático com HTML. Enfrentei desafios como estruturação semântica, integração com CSS e aplicação de boas práticas, que superei por meio de pesquisas, tutoriais e fóruns técnicos. O projeto foi fundamental para meu desenvolvimento profissional, permitindo que eu entregasse uma interface funcional e consolidasse conhecimentos essenciais em front-end.
